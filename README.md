# 🍷 Wine Festival Schedule

A simple and elegant event schedule webpage built using HTML and CSS.

## 🌐 Live Demo

👉 https://namratha669.github.io/wine-festival/

## ✨ Features
* 📅 Table-based event schedule
* 🎨 Styled using CSS
* 🔤 Custom fonts with Google Fonts
* 📐 Clean and centered layout


## 🛠️ Tech Stack

* HTML5
* CSS3

## 📚 What I Learned

* Structuring web pages using semantic HTML
* Creating tables with `<table>`, `<thead>`, `<tbody>`
* Styling layouts using CSS
* Organizing files in a project
